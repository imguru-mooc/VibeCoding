## 실습 3: 디버깅 실습 (14:00~14:30)

### 🔴 의도적 오류 수정 실습

아래 HTML 코드에는 **의도적으로 3개의 오류**가 있습니다.  
AI에게 오류를 찾아 수정해달라고 요청하세요.

```html
<!DOCTYPE html>
<html>
<head>
    <title>간단한 계산기</title>
    <style>
        body { font-family: Arial; text-align: center; margin-top: 50px; }
        input { padding: 10px; font-size: 18px; width: 80px; }
        button { padding: 10px 20px; font-size: 18px; cursor: pointer; }
        #result { font-size: 24px; margin-top: 20px; color: blue; }
    </style>
</head>
<body>
    <h1>간단한 계산기</h1>
    <input type="text" id="num1" placeholder="숫자1">
    <select id="operator">
        <option>+</option>
        <option>-</option>
        <option>×</option>
        <option>÷</option>
    </select>
    <input type="text" id="num2" placeholder="숫자2">
    <button onclick="calculate()">계산</button>
    <div id="result"></div>
    
    <script>
        function calculate() {
            let a = document.getElementById('num1').value;
            let b = document.getElementById('num2').value;
            let op = document.getElementById('operator').value;
            let answer;
            
            if (op === '+') answer = a + b;
            else if (op === '-') answer = a - b;
            else if (op === '×') answer = a * b;
            else if (op === '÷') answer = a / b;
            
            document.getElementById('result').innerHTML = '결과: ' + answer;
        }
    </script>
</body>
</html>
```

### AI에게 오류 수정 요청

```
아래 HTML 코드를 실행했더니 계산 결과가 이상하게 나와.
예를 들어 3 + 5를 하면 8이 아니라 "35"가 나와.
또한 0으로 나누기 처리가 안 돼.
코드를 검토하고 문제점을 찾아 수정해줘.
에러 원인도 설명해줘.

[위 코드 붙여넣기]
```

**기록:**

| 오류 | AI가 찾았는가? | 수정 방법 |
