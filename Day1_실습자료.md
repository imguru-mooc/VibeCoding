## 실습 3: AI로 업무 문서 작성하기 (14:00~14:40)

### 🟢 미션 3: 이메일 작성

아래 상황을 본인의 실제 업무로 바꿔서 AI에게 이메일 작성을 요청하세요.

**예시 프롬프트:**
```
다음 상황에 맞는 비즈니스 이메일을 작성해줘:

- 상황: 프로젝트 마감일이 1주일 연기되어 클라이언트에게 알려야 함
- 원인: 디자인 시안 수정 요청이 추가로 들어옴
- 대안: 중간 결과물을 먼저 공유하고, 최종본은 1주 후 전달
- 톤: 정중하지만 자신감 있게
- 길이: 200자 내외
- 형식: 인사 → 상황설명 → 대안제시 → 마무리
```

### 📝 나의 이메일 프롬프트 작성

```
상황: _______________________________________________
원인: _______________________________________________
대안: _______________________________________________
톤:   _______________________________________________
형식: _______________________________________________
```

**AI 결과물 평가:**
- [ ] 톤이 적절한가?
- [ ] 핵심 내용이 잘 전달되었는가?
- [ ] 바로 사용할 수 있는 수준인가?
- [ ] 수정이 필요한 부분은 무엇인가?

### 🟢 미션 4: 보고서 초안 작성

```
역할: [본인 직무] 담당자
맥락: [월간/주간] 업무 보고서 작성 (대상: [보고 대상])

이번 [주/달] 성과:
- [성과 1]
- [성과 2]
- [성과 3]

진행 중인 업무:
- [업무 1] (진행률: __%)
- [업무 2] (진행률: __%)

다음 [주/달] 계획:
- [계획 1]
- [계획 2]

형식: 요약 → 성과 → 진행현황 → 계획, 핵심 수치 볼드, A4 1장 분량
```
