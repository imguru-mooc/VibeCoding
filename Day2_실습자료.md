### 🟢 CRISP로 개선하기

위 3개의 나쁜 프롬프트를 CRISP 템플릿으로 개선하세요.

**CRISP 템플릿 복습:**
- **C** (Context): 맥락 — 배경, 상황, 업종, 팀
- **R** (Role): 역할 — AI에게 부여할 전문가 역할
- **I** (Instruction): 지시 — 구체적으로 해야 할 일
- **S** (Specifics): 세부사항 — 길이, 형식, 포함 항목
- **P** (Preference): 선호 — 톤, 출력 형식, 스타일

## 실습 2: 고급 프롬프트 기법 실습 (11:00~12:00)

### 🔵 기법 1: Few-Shot (예시 포함 프롬프트)

**미션:** 고객 리뷰 감성 분석기를 만드세요.

```
고객 리뷰의 감성을 분석해줘. 아래 예시를 참고해서 분석해줘.

예시:
- "배송이 정말 빨라서 좋았습니다" → 긍정 (이유: 배송 속도 만족)
- "상품이 사진과 너무 달라요" → 부정 (이유: 기대 불일치)
- "보통이에요 가격 대비 그저 그래요" → 중립 (이유: 특별한 감정 없음)

분석할 리뷰:
1. "포장이 꼼꼼하고 제품 상태도 완벽해요!"
2. "배송은 빨랐는데 상품에 흠집이 있네요"
3. "두 번째 구매인데 여전히 만족합니다"
4. "가격이 좀 비싸지만 품질은 확실해요"
5. "반품하고 싶은데 절차가 너무 복잡해요"

각 리뷰에 대해 감성(긍정/부정/중립), 이유, 개선 제안을 표로 정리해줘.
```

### 🔵 기법 2: Chain-of-Thought (단계적 추론)

**미션:** 아래 두 프롬프트를 같은 AI에 입력하고 결과 차이를 비교하세요.

**일반 프롬프트:**
```
우리 팀 매출이 전년 대비 15% 감소했어. 원인과 대응 방안을 알려줘.
```

**CoT 프롬프트:**
```
우리 팀 매출이 전년 대비 15% 감소했어. 단계별로 분석해줘:

1단계: 매출 감소의 가능한 외부 요인 3가지를 분석해줘
2단계: 가능한 내부 요인 3가지를 분석해줘
3단계: 각 요인별 구체적 대응 방안을 제시해줘
4단계: 대응 방안을 단기(1개월), 중기(3개월), 장기(6개월)로 나눠 정리해줘
5단계: 최우선 실행 항목 Top 3을 선정하고 이유를 설명해줘
```

**비교 기록:**

| 평가 항목 | 일반 프롬프트 (1~5) | CoT 프롬프트 (1~5) |
|----------|-----------------|-----------------|
| 분석 깊이 | | |
| 실행 가능성 | | |
| 구조화 수준 | | |
| 총점 | | |

### 🔵 기법 3: 역할극 (페르소나) 프롬프트

사업 아이디어: "AI를 활용한 중소기업 맞춤형 마케팅 자동화 서비스"

- 가격: 월 29만원

[역할]의 관점에서 장점 3가지, 우려사항 3가지, 개선 제안 2가지를 정리해줘.

1. CEO (전략적 가치 관점)
