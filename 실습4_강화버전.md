### 🎭 미션 B: AI에게 전문가 페르소나 부여하기 (15:15~15:30)

> AI는 "역할"을 줄수록 더 전문적인 답변을 제공합니다.  
> 같은 질문이라도 **일반 질문 vs 페르소나 부여** 시 얼마나 달라지는지 직접 비교해 보세요.

#### 실험 방법

**라운드 1 — 일반 질문** (페르소나 없이 그냥 물어보기)
```
신입사원을 교육하는 방법 알려줘.
```

**라운드 2 — 페르소나 부여 질문**
```
당신은 대기업 HR 팀에서 15년간 신입사원 온보딩을 설계해온 교육 전문가입니다.
현재 중소기업(직원 50명)에서 올해 처음으로 신입사원 3명을 채용했습니다.
별도 교육팀 없이 현업 팀장이 직접 교육해야 하는 상황에서
2주 안에 실무 투입 가능한 수준으로 만들기 위한 온보딩 플랜을 만들어주세요.
```

**비교 기록:**

| 비교 항목 | 라운드 1 (일반) | 라운드 2 (페르소나) |
|---------|-------------|-----------------|
| 답변 구체성 (1~5) | | |
| 현실 적용 가능성 (1~5) | | |
| 답변 길이/깊이 | | |
| 내가 바로 쓸 수 있는가? | ☐예 ☐아니오 | ☐예 ☐아니오 |

#### 나만의 페르소나 만들기
아래 빈칸을 채워 **내 업무에 최적화된 AI 페르소나**를 직접 설계해 보세요.

```
당신은 [업종/분야]에서 [경력]을 가진 [직무] 전문가입니다.
현재 [나의 회사/팀 상황]에서 [나의 직책]으로 일하고 있습니다.
[핵심 고민 또는 자주 묻는 질문 유형]에 대해 전문적인 조언을 제공해주세요.
```

✏️ 내가 작성한 페르소나:
```

현재 ______________________________________________________ 상황입니다.
___________________________________________________________에 대해 조언해주세요.
```

### ⚡ 미션 C: 프롬프트 리파이닝 챌린지 (15:30~15:50)

> 좋은 프롬프트는 한 번에 완성되지 않습니다.  
> **나쁜 프롬프트 → 개선 → 더 개선** 3단계로 직접 업그레이드해 보세요.

#### 📌 실험 시나리오
회사에서 다음 달 팀 워크숍을 기획해야 하는 상황입니다.

---

**1단계: 나쁜 프롬프트 (지금 바로 입력)**
```
팀 워크숍 기획해줘.
```
> AI 답변을 잠깐 읽고 아쉬운 점을 메모하세요.  
> 아쉬운 점: _____________________________________________

---

**2단계: 개선된 프롬프트 (조건 추가)**
```
15명 규모 팀의 1박 2일 워크숍을 기획해줘.
목적은 팀 결속력 강화이고 예산은 1인당 15만원이야.
장소는 수도권 근교로 해줘.
```
> AI 답변을 읽고 이전과의 차이를 메모하세요.  
> 달라진 점: _____________________________________________

---

**3단계: 최고의 프롬프트 (나만의 맥락 추가)**  
아래 빈칸을 채워 직접 완성해 보세요.
```
우리 팀은 [팀 특징: 예) 재택근무 중심, 평균 연령 30대 초반]이고,
최근 [팀 고민: 예) 부서간 소통 단절, 신규 입사자 적응 문제]이 있어.
이를 해결하는 데 도움이 되는 1박 2일 워크숍 프로그램을 짜줘.
예산은 1인당 15만원, 인원 15명, 수도권 근교 기준.
시간표 형식으로 정리하고, 각 프로그램의 기대 효과도 한 줄씩 써줘.
```

✏️ 내가 채운 최종 프롬프트:
```
우리 팀은 ________________이고, 최근 ________________ 문제가 있어.
이를 해결하는 데 도움이 되는 1박 2일 워크숍 프로그램을 짜줘.
예산은 1인당 ____만원, 인원 ____명, ______ 기준.
시간표 형식으로 정리하고, 각 프로그램의 기대 효과도 한 줄씩 써줘.
```

**3단계 비교:**

| | 1단계 | 2단계 | 3단계 |
|--|------|------|------|
| 내가 바로 쓸 수 있는가? | ☐예 ☐아니오 | ☐예 ☐아니오 | ☐예 ☐아니오 |
| 수정이 필요한 분량 | 많음/보통/적음 | 많음/보통/적음 | 많음/보통/적음 |

> 💬 **깨달음:** 프롬프트는 AI에게 보내는 "업무 지시서"입니다.  
> 지시가 구체적일수록, 결과물도 구체적입니다.

### 🤝 미션 D: AI 브레인스토밍 — 나만의 AI 활용 로드맵 만들기 (15:50~16:00)

> 지금까지 체험한 내용을 바탕으로, **향후 1개월 AI 도입 계획**을 AI와 함께 설계해 보세요.

**프롬프트:**
```
나는 [본인 직무]로 일하고 있어.
오늘 AI 도구(ChatGPT, Claude, Gemini)를 처음 체험해봤고,
[오늘 가장 인상 깊었던 기능]이 특히 유용할 것 같아.

내 업무 중 AI를 가장 먼저 적용하면 좋을 것 같은 영역은 [업무 영역]인데,
앞으로 1개월 동안 실천할 수 있는 AI 도입 로드맵을 만들어줘.

조건:
- 하루 30분 이내로 실천 가능한 수준
- 주차별 단계적 목표 포함
- 각 단계에서 사용할 AI 도구와 구체적인 프롬프트 예시 포함
```
